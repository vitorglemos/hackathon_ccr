# Jobb
Projeto desenvolvido para o hackathon ccr


## Nossa solução: 

Jobb é um aplicativo que une jovens e profissionais já alocados no mercado de trabalho para um bate papo online e/ou offline. Diminuímos o abismo entre a indecisão e a realização de um sonho. 

- Usando IA, recomendamos pessoas próximas de você para troca de experiências profissionais e acadêmicas.
- Você recebe notificação de eventos próximos de empresas cadastradas no Jobb.
- Marque um bate papo com alguém para discurirem sobre a área que você tanto sonha em seguir. Tire dúvidas com que já vivenciou isso.
